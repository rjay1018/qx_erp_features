📦 Smart Serial Inbound Scanner

Scan & Assign Serial Numbers with Ease During Incoming Shipments

This module allows warehouse teams to scan product serial numbers during incoming receipts using a barcode scanner or keyboard input. Each scanned serial number is automatically assigned to a product line, making the process fast, accurate, and efficient.

✅ Key Features

  • 📦 Add serial numbers on incoming shipments via scanning
  • 🧠 Automatically finds the next available move line
  • 🔁 Prevents duplicates and shows last scanned values
  • 📤 Fully compatible with barcode scanners or manual input
  • ⚡ Built for Odoo 14 Community — lightweight and fast

🔍 How It Works

  1. Go to Inventory › Receipts
  2. Select a picking with serial-tracked products
  3. Click on “Open Serial Scanner”
  4. Start scanning serials — they’re instantly assigned
  5. Review and validate when done
